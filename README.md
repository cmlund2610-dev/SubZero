# SubZero

SubZero is a React + Vite application focused on SaaS customer success analytics. It began from the standard Vite React starter and has been rebranded with a custom Joy UI theme, design tokens, and authentication flows.

Full project documentation, setup instructions, and brand guidelines will be added shortly.

## Stack
- React 19 + Vite 7
- Joy UI (MUI)
- Firebase (Auth/Firestore/Storage)
- Recharts
- ESLint 9

## Quick Start
Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

## Branding
Design tokens live in `src/assets/subzero.tokens.json` and the theme in `src/assets/theme.js`.
